# DMRL：面向多模态情感分析的反事实稀疏极性证据传输

## 摘要

多模态情感分析（Multimodal Sentiment Analysis, MSA）旨在通过融合文本、声学与视觉信号来推断人类情感。现有方法大多在推理之前就将异质线索压缩为单一的融合向量，这掩盖了情感证据的内部结构——不同线索之间可能相互**补充**、相互**冲突**，或在**可靠性**上存在差异。此外，模态重要性通常在融合阶段被一次性决定，下游预测无法对其进行修正；同时，融合权重很少与"每个组件究竟在多大程度上影响最终结果"这一原则性度量相绑定。为解决上述问题，我们提出 **DMRL**，一个基于**反事实稀疏极性证据传输**（Counterfactual Sparse Polar Evidence Transport）的框架。DMRL 首先通过混合静态—自适应槽注意力，从每个模态中提取少量稀疏的、具备极性与不确定性感知的证据槽；随后执行一种不确定性感知的 Sinkhorn 证据传输，以融合了语义、时序、极性与不确定性的复合代价对齐跨模态证据，并将匹配后的证据软分解为**一致（consensus）**、**冲突（conflict）**与**残差（residual）**三种角色；最后，一个反事实可靠性路由器对模态和角色进行加权，其监督信号来自"消融某一组件时预测发生的变化量"，同时一个共享的序数分布预测头给出情感估计与校准后的预测方差。在标准 MSA 基准上的大量实验表明，DMRL 在取得有竞争力性能的同时，提升了跨模态情感证据的可解释性。

**关键词**：多模态情感分析，最优传输，反事实推理，证据分解，不确定性估计

## 1. 引言

多模态情感分析（MSA）旨在通过联合建模在对话视频中共现的语言、声学与视觉流来理解人类情感。由于非语言线索常常能消解、甚至反转口语词汇所表现出的表面极性，有效的跨模态融合是该任务的核心。

尽管已有大量进展，仍存在三个局限：

1. **证据结构的过早坍缩。** 许多模型在进行情感推理之前，就将所有模态聚合为单一紧凑表示。这种早期聚合会把扮演本质不同角色的线索纠缠在一起：有些线索相互增强，有些直接冲突，还有些信息量微弱或带有噪声。一旦坍缩，这些区分便无法再被恢复或推理。

2. **模态贡献无法被下游修正。** 模态贡献通常只在融合阶段被估计一次，使得后续推理无法根据各模态彼此之间的关系来修正它们。

3. **融合权重缺乏因果依据。** 分配给各模态的权重很少建立在"每个组件对最终预测改变多少"这一显式且可度量的概念之上；相反，它们由难以解释、且容易抓住虚假捷径（最典型的是对文本模态的过度依赖）的不透明注意力分数产生。

我们认为，对 MSA 更忠实的处理应当：(i) 保留情感证据的内部结构，而非过早将其坍缩；(ii) 显式刻画证据在跨模态间所扮演的角色；(iii) 将融合权重与一种因果的、反事实的重要性度量以及校准后的不确定性相绑定。基于这些原则，我们提出 **DMRL**，一个基于**反事实稀疏极性证据传输**的框架。

DMRL 通过模态专属的多尺度卷积 stem、共享 Transformer 编码器与轻量级模态适配器，将每个模态映射到共享的编码空间。在编码后的序列上，它为每个模态提取少量**稀疏极性证据槽**；每个槽携带一个证据向量以及可解释的标量属性：极性、强度、不确定性和时序位置。槽的可靠性被定义为其强度减去其不确定性，可靠性同时驱动池化与路由。随后，跨模态证据通过一种**不确定性感知的 Sinkhorn 传输**进行匹配，其代价融合了特征距离、时序距离、极性距离以及不确定性惩罚。匹配后的证据依据语义与极性上的一致程度被软分解为**一致**、**冲突**与**残差**三个库。一个**反事实可靠性路由器**同时产出模态权重与角色权重；关键在于，它由"消融某一模态或角色时所观测到的预测变化幅度"来监督，使得越重要的组件获得越大的权重。预测由一个共享的**序数分布预测头**在固定情感锚点上完成，其期望给出情感分数，其方差提供校准后的预测不确定性。

本文的主要贡献如下：

- 我们提出一种稀疏极性证据表示，显式暴露每个模态中少数决定性线索的极性、强度、不确定性与时序位置，为下游推理保留了证据结构。
- 我们提出一种不确定性感知的 Sinkhorn 证据传输，利用复合代价对齐跨模态证据，并将匹配后的证据软分解为一致、冲突、残差三种角色。
- 我们设计了一个反事实可靠性路由器，将模态与角色权重建立在"消融每个组件所引起的预测变化"之上，并在事实预测与反事实预测之间共享同一个序数预测头，以保证比较的公平性。
- 我们在标准 MSA 基准上进行了大量实验与分析，表明 DMRL 在与强基线相当的同时，能够产出可解释的证据与校准的不确定性。

## 2. 相关工作

### 2.1 多模态情感分析

早期 MSA 研究聚焦于通过张量融合 [4]、注意力 [5] 以及模态翻译目标来建模跨模态交互。一条重要的研究路线是将表示分解为共享子空间与模态专属（私有）子空间，以控制冗余并保留判别性线索，其代表是 MISA [6]。近期方法进一步细化这一分解：FINE [16] 通过互信息估计来因子化共享与独有表示并抑制任务无关噪声；TSD [17] 在全局共享与私有之外引入成对共享子空间以建模成对协同；DLF [18] 则以语言为中心解耦共享/专属信息并通过几何约束精化；DERL [19] 则采用解耦专家以在模态缺失时保持鲁棒。其中，PRISM [9] 将多模态证据组织在共享原型空间中，显式建模证据之间的互补、冲突与可靠性差异，并在推理过程中对模态贡献进行动态重加权，其动机与本文最为接近。然而，上述方法多依赖原型空间中的隐式结构或“共享/私有”的静态分解，而 DMRL 与子空间分解及原型方法的不同之处在于：它并非沿"共享/私有"轴来组织证据，而是沿通过跨模态最优传输匹配显式定义的功能性**角色**（一致/冲突/残差）来组织，并且保持多个携带极性、强度、不确定性与时序属性的决定性证据槽显式存在，而非产出单一的逐模态向量或原型。

### 2.2 用于跨模态对齐的最优传输

最优传输（OT）及其熵正则化的 Sinkhorn 近似 [7]，为两组元素之间的软匹配提供了原则性机制。在 MSA 中，PaSE [10] 利用熵正则 OT 对齐原型表示，并结合基于 Shapley 的梯度调制来缓解模态竞争，是将 OT 引入 MSA 的代表性近期工作。我们的传输模块与之不同：PaSE 在原型层面以特征距离为主导进行对齐，而 DMRL 在少量证据槽上操作，并使用一种复合传输代价，同时考虑语义相似度、时序邻近度、极性一致性以及被匹配槽的不确定性，而非仅依赖特征距离；这使得传输计划本身即可区分一致与冲突的跨模态证据。

### 2.3 MSA 中的反事实与因果推理

反事实与因果推理在 MSA 中主要被用于消除单模态偏差，最常见的是文本词汇与标签之间的虚假相关。这类方法构建因果图，通过反事实输入估计某一模态的直接效应，并将其减去以获得去偏预测，如 CF-MSA [11]、MCIS [12] 与面向分布外泛化的反事实推理 [13]。近期工作进一步将因果干预与信息瓶颈相结合：MMCI [20] 将输入建模为多关系图并通过后门调整分离模态内/间的因果与捆绑特征，CaMIB [21] 则借助信息瓶颈与工具变量约束学习最小因果表示以提升分布外泛化。与之相关地，MCAF [14] 通过基于信息瓶颈的因果干预解耦语言模态的情感偏差，并设计动态路由器实时评估视觉、声学与去混淆语言信号之间的"互补/冲突/冗余"交互状态以分配权重，其角色化与动态路由的思路与本文高度相关。DMRL 将反事实用于不同目的：它并非用于去偏或仅做交互状态分类，而是将"消融每个模态、每个角色时的**预测变化幅度**"作为可靠性路由器的直接监督信号，并在事实与反事实预测间共享同一序数头，从而把反事实敏感度转化为一个显式的、可学习的组件重要性度量。

### 2.4 不确定性与序数建模

可信多模态学习通过证据式或高斯混合等形式估计预测不确定性，以实现自适应的、可靠性感知的融合，如基于正态-逆伽马混合分布的可信多模态回归 [8]；近期工作还将不确定性感知融合推广到联邦与可靠性加权聚合场景 [22]。在 MSA 中，TMSON [15] 为每个模态估计不确定性分布，通过贝叶斯融合获得多模态分布，并在序数情感空间中以序数回归约束预测，是同时利用不确定性与序数有序性的代表性工作。DMRL 在单一预测头中结合了这两种视角，但形式更为紧凑：它直接预测一个在序数情感锚点上的分布，并以其期望作为分数、以其方差作为校准后的不确定性，从而避免了 TMSON 中分离的不确定性分布估计与贝叶斯融合步骤，也无需额外的区间预测分支。

## 3. 方法

给定文本序列 $\mathbf{X}_t$、声学序列 $\mathbf{X}_a$ 与视觉序列 $\mathbf{X}_v$，DMRL 依次经过：(i) 带模态适配器的共享编码；(ii) 稀疏极性证据提取；(iii) 不确定性感知的 Sinkhorn 证据传输与软角色分解；(iv) 反事实可靠性路由；(v) 序数分布预测。

### 3.1 带模态适配器的共享编码

对于文本，使用预训练的 BERT/RoBERTa 编码器产生 token 级特征。每个模态 $m \in \{t,a,v\}$ 首先经过一个由并行 $1$、$3$、$5$ 卷积核的一维卷积构成、并带残差投影的多尺度时序 stem，将所有模态映射到共同的 $d$ 维空间：

$$\mathbf{H}_m^{0} = \mathrm{Stem}_m(\mathbf{X}_m) + \mathbf{e}_m,$$

其中 $\mathbf{e}_m$ 是可学习的模态嵌入。随后一个**共享**的、掩码感知的 Transformer 编码器 $\mathcal{E}$ 处理所有模态（这能减少参数量并促进形成共同的证据空间），其后接一个轻量级的模态专属残差适配器 $\mathcal{A}_m$：

$$\mathbf{H}_m = \mathcal{A}_m\big(\mathcal{E}(\mathbf{H}_m^{0})\big).$$

### 3.2 稀疏极性证据提取

从 $\mathbf{H}_m \in \mathbb{R}^{T_m \times d}$ 出发，我们通过带**混合静态—自适应查询**的槽注意力提取 $K$ 个证据槽。静态槽查询 $\mathbf{Q}^{0}\in\mathbb{R}^{K\times d}$ 通过门控函数被全局上下文 $\mathbf{c}_m = \mathrm{MaskedMean}(\mathbf{H}_m)$ 调制：

$$\mathbf{Q} = \mathrm{LN}\!\left(\mathbf{Q}^{0} + \bm{g}\odot \phi(\mathbf{c}_m)\right), \qquad \bm{g} = \sigma\!\left(\mathrm{MLP}([\mathbf{Q}^{0};\phi(\mathbf{c}_m)])\right).$$

注意力分数可选地由时序轴上的 top-$k$ 算子稀疏化，再经掩码 softmax，得到 $\mathbf{A}\in\mathbb{R}^{K\times T_m}$，证据向量为 $\mathbf{Z} = \mathbf{A}\,\mathbf{V}(\mathbf{H}_m)$。每个槽 $i$ 通过小型预测头被赋予可解释属性：

$$p_i = \tanh(\cdot)\in[-1,1] \ \text{（极性）}, \quad s_i = \mathrm{softplus}(\cdot)\ge 0 \ \text{（强度）},$$

$$u_i = \mathrm{softplus}(\cdot)\ge 0 \ \text{（不确定性）}, \quad \tau_i = 0.5\textstyle\sum_t \mathbf{A}_{it}g_t + 0.5\sigma(\cdot)\in[0,1] \ \text{（位置）},$$

其中 $g_t$ 是归一化的时序网格。槽的**可靠性**为 $r_i = s_i - u_i$，池化权重为 $\bm{w} = \mathrm{softmax}(\bm{r})$，因此模态摘要为 $\bar{\mathbf{z}}_m = \sum_i w_i \mathbf{z}_i$。一个槽相似度矩阵 $\mathbf{S}_m = \hat{\mathbf{Z}}\hat{\mathbf{Z}}^{\top}$ 被保留，用于多样性正则。

### 3.3 不确定性感知的 Sinkhorn 证据传输

对一个有序模态对 $(a,b)$，我们用熵正则 OT 计划匹配其证据槽。令 $\mathrm{sim}_{ij} = \hat{\mathbf{z}}^{a}_i \cdot \hat{\mathbf{z}}^{b}_j$，传输代价融合四项：

$$C_{ij} = \lambda_f (1-\mathrm{sim}_{ij}) + \lambda_t |\tau^a_i - \tau^b_j| + \lambda_p |p^a_i - p^b_j| + \lambda_u \tfrac{1}{2}(u^a_i + u^b_j).$$

传输计划通过对 $\log\bm{\alpha} = -C/\epsilon$ 执行 Sinkhorn 归一化得到：

$$\mathbf{T} = \mathrm{Sinkhorn}\!\left(-C/\epsilon\right),$$

即交替的行/列 log-sum-exp 归一化，最后做一次全局归一化。传输计划 $\mathbf{T}$、代价 $C$ 与相似度 $\mathrm{sim}$ 一并传入角色分解。

### 3.4 软一致—冲突—残差分解

给定行归一化的传输 $\tilde{\mathbf{T}}$，模态 $a$ 中的每个槽 $i$ 与一个被传输的对应物 $\tilde{\mathbf{z}}^{b}_i = \sum_j \tilde{T}_{ij}\mathbf{z}^b_j$ 配对，被传输的极性 $\tilde{p}^b_i$ 与不确定性 $\tilde{u}^b_i$ 同理获得。我们定义平均对齐 $\bar{s}_i = \sum_j \tilde{T}_{ij}\mathrm{sim}_{ij}$、极性差 $\delta_i = |p^a_i - \tilde{p}^b_i|$ 与配对不确定性 $\bar{u}_i = \tfrac{1}{2}(u^a_i + \tilde{u}^b_i)$，并计算三个角色分数：

$$\text{cons}_i = \bar{s}_i - \lambda_p\delta_i - \lambda_u\bar{u}_i, \quad \text{conf}_i = \bar{s}_i + \lambda_p\delta_i - \lambda_u\bar{u}_i, \quad \text{res}_i = -\bar{s}_i + \lambda_u\bar{u}_i,$$

并经带温度的 softmax 得到角色概率 $\bm{\pi}_i=(\pi^{c}_i,\pi^{f}_i,\pi^{r}_i)$。角色专属证据为：

$$\mathbf{c}_i = \pi^{c}_i\cdot\tfrac{1}{2}(\mathbf{z}^a_i+\tilde{\mathbf{z}}^b_i), \quad \mathbf{f}_i = \pi^{f}_i\cdot(\mathbf{z}^a_i-\tilde{\mathbf{z}}^b_i), \quad \mathbf{r}_i = \pi^{r}_i\cdot\mathbf{z}^a_i,$$

即：一致角色聚合对齐且同极性的证据；冲突角色捕捉对齐但极性相反证据的带符号差异；残差角色保留弱对齐或不确定的证据。每个模态对（文本—音频、文本—视频，可选音频—视频）均双向计算角色，拼接成一致、冲突、残差三个库并投影。每个库按其能量池化，并由专属的角色专家处理，产出 $\mathbf{c},\mathbf{f},\mathbf{r}\in\mathbb{R}^d$。

### 3.5 反事实可靠性路由器

两个可靠性信号驱动路由。模态不确定性为可靠性加权的槽不确定性 $U^{\text{mod}}_m = \sum_i w^m_i u^m_i$，角色不确定性为角色专家的能量归一化逆幅度代理。路由器输出模态与角色权重，不确定性直接抑制对应 logit：

$$\bm{\ell}^{\text{mod}} = \mathrm{MLP}([\bar{\mathbf{z}}_t;\bar{\mathbf{z}}_a;\bar{\mathbf{z}}_v;\mathbf{U}^{\text{mod}}]) - \beta_m\mathbf{U}^{\text{mod}},$$

$$\bm{\ell}^{\text{role}} = \mathrm{MLP}([\mathbf{c};\mathbf{f};\mathbf{r};\mathbf{U}^{\text{role}}]) - \beta_r\mathbf{U}^{\text{role}},$$

其中 $\bm{m}=\mathrm{softmax}(\bm{\ell}^{\text{mod}})$，$\bm{\rho}=\mathrm{softmax}(\bm{\ell}^{\text{role}})$。

**反事实监督。** 令 $\hat{y}$ 表示由共享序数头得到的事实预测。对每个模态 $m$，我们通过将其池化表示置零、同时保持其它输入与同一预测头不变，得到反事实预测 $\hat{y}_{\neg m}$；角色反事实 $\hat{y}_{\neg \cdot}$ 通过将某角色表示置零类似地构造。反事实目标为：

$$\bm{t}^{\text{mod}} = \mathrm{softmax}\!\Big(\tfrac{1}{\tau}\big[\,|\hat{y}-\hat{y}_{\neg t}|,\,|\hat{y}-\hat{y}_{\neg a}|,\,|\hat{y}-\hat{y}_{\neg v}|\,\big]\Big),$$

$\bm{t}^{\text{role}}$ 同理。路由器被训练以匹配这些目标（见 3.7 节），使得移除后预测变化更大的组件被分配更大的权重。在事实与反事实预测之间共享同一个序数头，确保该比较不会被一个独立的预测器所混淆。

### 3.6 序数分布预测头

模态表示与角色表示被融合为单一向量 $\mathbf{h} = \mathrm{LN}(\mathbf{h}^{\text{mod}} + \mathbf{h}^{\text{role}})$，其中 $\mathbf{h}^{\text{mod}}$ 用 $\bm{m}$ 加权三个池化模态，$\mathbf{h}^{\text{role}}$ 用 $\bm{\rho}$ 加权三个角色。在固定锚点 $\bm{a}=(-3,-2,-1,0,1,2,3)$ 上，预测头产出类别分布 $\bm{q}=\mathrm{softmax}(\mathrm{MLP}(\mathbf{h}))$，并报告：

$$\hat{y} = \sum_k q_k a_k, \qquad \mathrm{Var} = \sum_k q_k (a_k - \hat{y})^2,$$

以 $\hat{y}$ 作为情感分数、$\mathrm{Var}$ 作为预测不确定性。

### 3.7 训练目标

DMRL 通过若干互补项的加权和进行优化。主回归损失与辅助回归损失使用 $L_1$：

$$\mathcal{L}_{\text{main}} = |\hat{y}-y|, \quad \mathcal{L}_{\text{uni}} = \tfrac{1}{3}\!\!\sum_{m\in\{t,a,v\}}\!\!|\hat{y}_m - y|, \quad \mathcal{L}_{\text{role}} = \tfrac{1}{3}\!\!\sum_{o\in\{c,f,r\}}\!\!|\hat{y}_o - y|.$$

序数损失是在锚点上以高斯平滑目标计算的软交叉熵。稀疏正则 $\mathcal{L}_{\text{sparse}}$ 结合了低注意力熵、低槽权重熵以及低非对角槽相似度，以鼓励少而决定性且多样的证据槽。传输损失 $\mathcal{L}_{\text{trans}}=\sum_{\text{pairs}}\langle\mathbf{T},C\rangle$ 最小化匹配代价。反事实路由损失将路由权重与反事实目标对齐：

$$\mathcal{L}_{\text{cf}} = \tfrac{1}{2}\big[\mathrm{KL}(\bm{t}^{\text{mod}}\Vert\bm{m}) + \mathrm{KL}(\bm{t}^{\text{role}}\Vert\bm{\rho})\big].$$

最后，方差校准损失将预测方差匹配到实际平方误差，$\mathcal{L}_{\text{var}}=|\mathrm{Var}-(\hat{y}-y)^2|$（目标做 detach）。总目标为：

$$\mathcal{L} = w_1\mathcal{L}_{\text{main}} + w_2\mathcal{L}_{\text{uni}} + w_3\mathcal{L}_{\text{role}} + w_4\mathcal{L}_{\text{ord}} + w_5\mathcal{L}_{\text{sparse}} + w_6\mathcal{L}_{\text{trans}} + w_7\mathcal{L}_{\text{cf}} + w_8\mathcal{L}_{\text{var}}.$$

---

## 4. 实验

### 4.1 数据集与评测指标

我们在标准的词对齐 MSA 基准 CMU-MOSI、CMU-MOSEI 与 CH-SIMS 上进行评估。遵循通用做法，我们报告二分类准确率（Acc-2）、七分类准确率（Acc-7）、F1 分数、平均绝对误差（MAE）与皮尔逊相关系数（Corr）。对 Acc-2 与 F1，在适用处同时报告"负/非负"与"负/正"两种约定。

### 4.2 实现细节

文本特征由预训练 BERT（或 RoBERTa）编码器提取；声学与视觉特征遵循各数据集的标准流程。我们设模型宽度 $d{=}128$，共享编码器层数为 $3$，注意力头数为 $4$，每模态证据槽数 $K{=}4$。Sinkhorn 温度 $\epsilon{=}0.2$，迭代 $5$ 次，角色温度为 $0.5$。损失权重为 $(w_1,\dots,w_8)=(1.0,0.2,0.1,0.2,0.01,0.02,0.05,0.005)$。模型使用 AdamW 训练，并以验证集 MAE 做早停。所有报告数值均为多个随机种子的平均。

### 4.5 分析

**反事实路由。** 我们可视化路由权重与所测得的反事实预测变化之间的一致性，表明路由器学会了对"消融后最影响结果"的组件赋予更高权重。

**证据可解释性。** 可逐条样本检视证据槽的极性、强度与时序位置，揭示决定性情感线索在时间上出现的位置，以及模态之间如何一致或冲突。

**不确定性校准。** 预测方差与实际误差相关，表明 DMRL 能产出校准的置信度估计，可支持选择性预测。

---

## 5. 结论

我们提出了 DMRL，一个多模态情感分析框架：它通过稀疏极性证据槽保留情感证据的结构，以不确定性感知的 Sinkhorn 传输对齐跨模态线索，将匹配后的证据分解为一致/冲突/残差角色，并将融合权重建立在组件重要性的反事实度量之上，同时通过共享的序数分布预测头报告校准的不确定性。实验与分析表明，DMRL 与强基线相当，并提供了更好的可解释性。未来工作包括：将传输扩展到三个以上模态、通过同一证据抽象处理模态缺失，以及研究反事实监督路由的理论性质。
